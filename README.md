# Calculadora de Horas Trabalhadas

Uma ferramenta simples para calcular o total de horas trabalhadas com base nos horários de entrada e saída.

## Como usar

1. Acesse a [calculadora](https://mozartsempiano.github.io/calculadora-horas-trabalhadas/).
2. Informe os campos necessários e horários de entrada e saída em cada dia do ciclo mensal.
3. A calculadora soma o período total trabalhado.
4. Exporte seus dados como backup.

## Recursos

- Cálculo automático do total de horas trabalhadas, horas previstas, saldo e horas/dia restantes.
- Interface simples e intuitiva.
- Pode ser utilizada para controle de ponto diário ou mensal.
